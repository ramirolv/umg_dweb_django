Tecnologías y herramientas utilizadas

HTML

Es el lenguaje que ayuda a formar la maquetación del sitio web.

CSS

Es el lenguaje de hojas de estilos en cascada que permite definir los estilos visuales.

JavaScript

Es el lenguaje de programación que ayuda a codificar la interactividad del sitio.

Bootstrap

Kit de herramientas FrontEnd, ampliable y repleto de funciones.

VS Code

Editor de código versátil y poderoso por sus extensiones y atajos.

Git

Software de control de versiones.

GitHub

Plataforma web para compartir y almacenar repositorios de código.

MDN Web Docs

Documentación sobre estándares de tecnologías web impulsado por Mozilla
